# Session 01: Elements and Console

This session introduces the two DevTools panels used most often during frontend debugging:

- **Elements**: inspect HTML, edit CSS, debug layout, review accessibility, and understand event listeners.
- **Console**: run JavaScript, read logs, filter messages, inspect selected elements, and understand errors or warnings.

Use the detailed notes and demo files inside each folder for hands-on practice.

## Open DevTools

| Method | Windows/Linux | macOS |
|---|---|---|
| Open DevTools | `F12` or `Ctrl+Shift+I` | `Cmd+Option+I` |
| Open Console directly | `Ctrl+Shift+J` | `Cmd+Option+J` |
| Pick an element | `Ctrl+Shift+C` | `Cmd+Shift+C` |
| Command Menu | `Ctrl+Shift+P` | `Cmd+Shift+P` |

You can also right-click any page element and select **Inspect**.
