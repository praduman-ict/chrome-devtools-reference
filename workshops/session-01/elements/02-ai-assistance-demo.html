Elements AI Assistance Demo

Select one problem element in the Elements panel, right-click it, then choose Ask AI. The selected node becomes the context for the chat.

How to Practice

  1. Open DevTools and go to the Elements panel.
  2. Select one highlighted problem area below.
  3. Right-click the selected DOM node and choose Ask AI.
  4. Ask a specific question about layout, styling, or accessibility.
  5. Test any suggested CSS changes in the Styles panel before saving them.

1. Overflow Problem

Inspect .wide-content or its parent .overflow-demo.

This child is intentionally wider than its parent. Ask AI why the content is clipped and what CSS changes could make it responsive.
  • Why is this content overflowing or clipped?
  • What CSS change would make this section responsive?

2. Flex Layout Problem

Inspect .layout-demo and ask why the children do not fit cleanly.

Main content area set to 72% width.
Sidebar set to 44% width.
  • Why are these flex children too wide for the row?
  • Suggest a safer flex layout for this component.

3. Contrast Problem

Inspect the paragraph and ask AI to explain the readability issue.

This text intentionally has weak contrast against the background.

  • Does this text have enough contrast?
  • Suggest a more accessible text color.

4. Spacing Problem

Inspect the grid or one button and ask AI about spacing consistency.

  • Are these buttons spaced consistently?
  • What CSS controls the spacing here?
